---
title: Upgrade to Galaxy version 25.0
tags: [upgrade]
---

The latest Galaxy version 25.0 brings many new features to useGalaxy.be, including:

- Empowing users to build more kinds of collections, more intelligently
- Added ZIP explorer functionality to import individual files from local or remote ZIP archives
- A new unified view for datasets
- Rerunning workflows with the same inputs and parameters 
- An enhanced workflow run form interface
- Bulk actions for workflows
- Introducing READMEs for workflows in the Galaxy interface
- JupyterLite for lightweight in-browser notebooks within Galaxy
- Preferred visualization display for dataset types
- And more!

For more details, visit [the official Galaxy project release v25.0 post](https://docs.galaxyproject.org/en/master/releases/25.0_announce_user.html). 

Please report any (new) issues you are experiencing via <a href="mailto:datacore.galaxy@vib.be">datacore.galaxy@vib.be</a>.
