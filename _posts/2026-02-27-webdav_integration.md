---
title: 'WebDAV protocol (e.g. Nextcloud) file source integration'
---

You can now make use of remote file sources that use the WebDAV protocol such as Nextcloud, Owncloud, Opencloud,... as a remote file source.

Browse to <a href="https://usegalaxy.be/file_source_instances/create">Remote file sources</a> in your user settings to set this up. 

If you're experiencing problems setting up such connection, please reach out to our support mail at <a href="mailto:datacore.galaxy@vib.be">datacore.galaxy@vib.be</a>.
