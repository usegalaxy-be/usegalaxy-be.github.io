---
title: 'General improvements to our tool offer'
---

Recently, some general improvements were made to the current tool offer of useGalaxy.be:

- All installed tools are automatically updated every week to their latest available version. You can still use an older tool version through the *versions* button, found in the top right corner of the tool page (left of the *Run tool* button). 

- Tools with a failing tool test have been manually fixed. We encourage you to resubmit jobs that were not successful in the past and kindly ask you to submit a bug report if you still end up with a failed job. Make sure you're on the latest version of the tool, as older tool versions might be deprecated but are not removed for reproducibility. 

- Tools and/or specific tool versions that are deprecated were hidden from the tool panel. These tools can still be found and used when they are part of an older workflow or history, but are not easily findable anymore through the tool panel. If you would still require such a tool, please reach out via <a href="mailto:datacore.galaxy@vib.be">datacore.galaxy@vib.be</a>.
