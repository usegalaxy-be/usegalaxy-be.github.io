---
title: Upgrade to Galaxy version 26.0
tags: [upgrade]
---

UseGalaxy.be has been upgraded to 26.0, which comes with bug fixes and new features from both v25.1 and v26.0:

- ChatGXY: intelligent assistance inside Galaxy
- AI assisted notebooks with JupyterLite
- AI powered data visualization with Vintent
- Automated actions when workflows finish
- Improved workflow error navigation
- Enhanced tool discovery view
- Smart sample sheets for complex workflow inputs
- Credential definition in tools
- Workflow editor improvements
- Galaxy Charts framework, which for example brings the IGV.js Genome Browser
- And more!

For more details, visit the official post of The Galaxy Project on [release v25.1](https://docs.galaxyproject.org/en/master/releases/25.1_announce_user.html) and [release v26.0](https://docs.galaxyproject.org/en/master/releases/26.0_announce_user.html).

Please report any (new) issues you are experiencing to us via: 'datacore.galaxy (at) vib.be'.
