Galaxy Community Conference - a brief summary!

 August 6, 2019 Belgium training tiaas gcc

The 2019 Galaxy Community Conference (GCC2019) was held in Freiburg, Germany, 1-8 July. 2019 was the largest GCC ever with 231 participants, 39 Galaxy training courses, 65 talks, 54 posters, 13 demos, and more than 30 different BoFs. Our participants came from 27 different countries of Eastern- and Western Europe, North America, Asia, Oceania and Sub-saharan Africa.

MaxQuant goes Galaxy

 August 2, 2019 Belgium tools

The Belgian Galaxy server now contains the latest MaxQuant tool wrapper. One featuring options for SILAC and label free quantification and also the possibility to process fractioned data by uploading an experimental design template.

PLAZA reference data powering UseGalaxy.be

 July 1, 2019 Belgium tools gcc

Reference data for Plant species at UseGalaxy.be is directly streamed from PLAZA data warehouse. If you want to install this set of reference data directly into your Galaxy instance check out this repo. To handle other (non-plants) reference data a set of helpful scripts can be found here. This work was presented during the 2019 Galaxy Community Conference (GCC2019) held in Freiburg, Germany, 1-8 July. Take a look at the poster.

Welcome to UseGalaxy.be: The Belgian Galaxy instance

 June 1, 2019 Belgium tools elixir

Welcome to the Belgian Galaxy instance. Relevant code used to deploy and maintain this instance, together with associated documentation can be found here. Registration is free and open to anyone, you will get a total of 50GB to use on your own analysis. You can also explore and test the instance without the need to register, for this you get 5GB of space for data.
