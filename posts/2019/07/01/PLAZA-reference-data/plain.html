PLAZA reference data powering UseGalaxy.be

»
Jul 1, 2019
Belgium tools gcc

Reference data for Plant species at UseGalaxy.be is directly streamed from PLAZA data warehouse. If you want to install this set of reference data directly into your Galaxy instance check out this repo. To handle other (non-plants) reference data a set of helpful scripts can be found here. This work was presented during the 2019 Galaxy Community Conference (GCC2019) held in Freiburg, Germany, 1-8 July. Take a look at the poster.
